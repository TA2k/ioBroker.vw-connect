Tibber Data API (optional, nur für VW ID)
Tibber liefert über deren Data API SoC, Range und Lade-Status für VW/Audi/Škoda/Cupra Fahrzeuge.

Voraussetzungen:
  1. Tibber-Account anlegen (geht NUR über die Tibber-App, nicht über die Webseite). Ein Tibber-Stromvertrag ist NICHT nötig.
  2. In der Tibber-App das Fahrzeug verbinden (Connect Vehicle).
  3. Im Browser https://data-api.tibber.com/clients/manage öffnen, mit dem gleichen Tibber-Account einloggen, OAuth2-Client anlegen mit:
    • Name: frei wählbar (z.B. "ioBroker"); darf NICHT mit "tibber" beginnen.
    • Scopes: alle anhaken (Tibber verlangt mindestens einen zusätzlichen Scope; alles ankreuzen ist sicher).
    • Redirect URI: http://localhost/ (mit Slash am Ende, exakte Schreibweise!).
  4. Client ID und Client Secret hier unten einfügen.
  5. Den Adapter speichern. Im Adapter erscheint dann oben ein Authorize-Link — diesen im Browser öffnen, mit Tibber einloggen, auf der Berechtigungs-Seite alle Häkchen lassen und unten "Yes, allow" klicken. Nach dem Klick landest du auf einer leeren http://localhost/?code=...&state=...-Seite (Browser zeigt Fehler — egal). Die komplette URL aus der Adressleiste (mit allem nach dem ?) hier ins "Tibber Authorize Callback-URL"-Feld einfügen und nochmal speichern.
  6. Adapter tauscht den Code gegen einen Refresh-Token, der danach automatisch rotiert.
Authorize-URL erscheint hier sobald die Client ID oben eingetragen ist.
